# 90〜00年代風 自己紹介サイト 実装計画

Github Pagesでホストして公開するための、90年代後半〜2000年代初頭の「タグ打ち風」レトロ自己紹介サイトを作成します。

## 目標
当時の個人サイト（Geocitiesなどにあったようなサイト）の独特な文化と見た目を再現します。
ビジュアルの「ダサさ」「懐かしさ」を最大限に引き出します。

## ユーザー確認事項
> [!NOTE]
> GitHub Pagesへのデプロイについては、最終的に完成したディレクトリ群をユーザーの方でGitHubにPushしていただき、設定を行っていただく想定です。本タスクではローカルで動作するHTML/CSS一式を作成します。

> [!TIP]
> 以下の「あるある」要素を盛り立たせようと考えていますが、特に取り入れたい（あるいは外したい）要素はありますでしょうか？
> - `<marquee>`タグ（流れる文字）
> - チカチカ点滅する文字（CSSのblinkアニメーションで再現）
> - カラフルで目に厳しいテキストカラーと、黒または星空の背景
> - キリ番（キリのいい訪問者数）報告の要求
> - 「あなたは〇〇人目の訪問者です」アクセスカウンター（ダミー画像またはJSランダム）
> - 工事中（Under Construction）のGIFとテキスト
> - 「IE5.0以上、解像度800x600推奨」といったバナー
> - **※見た目はレトロにしつつ、スマホ等でも見やすいレスポンシブ対応を行います。**
