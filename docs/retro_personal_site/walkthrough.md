# Walkthrough: 90〜00年代風 個人サイト

## 実装した内容

GitHub Pagesで公開するための90〜00年代風（懐かしのタグ打ち＆テーブル風）自己紹介サイトを一式、以下の内容で作成しました。

- **`index.html` (玄関/トップページ)**
  - CSSアニメーションによる `<marquee>` (流れる文字) と `<blink>` (点滅文字) の再現
  - グラデーションによるレインボーテキスト装飾（†漆黒の堕天使の館†）
  - Javascriptによるランダムな「なんちゃってアクセスカウンター」
  - キリ番報告要求の文面
  - 「戻る」ボタンやメニュー表示など
- **`profile.html` (自己紹介ページ)**
  - CSSで罫線を太く表現した、無骨な自己紹介テーブル
  - 当時流行した「◯◯の質問」の一部再現
- **`diary.html` (日記/ポエムページ)**
  - 黒背景・ライム/黄色文字などを使った目に痛い配色
  - 昔ながらの「つぶやき・ポエム」レイアウト
- **`link.html` (リンク・同盟ページ)**
  - バナー画像が無いことを想定した、CSSグラデーションによる「ダミーバナー」
  - 相互リンクのお願い（直リンク禁止、お持ち帰り推奨の文言）
- **`style.css` (スタイルシート)**
  - `word-break` や `max-width` 等を用いて、基本は中央寄せの固定幅風に見せつつ、**スマートフォンで開いた際は画面幅に収まるレスポンシブ対応**を行いました。
  - 背景画像が見つからなかった場合でも「それっぽく」見えるよう、CSSのグラデーション(`radial-gradient`)を用いて星空風の背景を描画しています。

## 検証結果

- ローカルにて `index.html` を開くことで、以下が期待通りに動作・表示されることを確認できます。
  - 画面サイズの変更時（スマホサイズ）に、メニューが縦並びになり、テーブル要素がブロック化されて読みやすくなること（レスポンシブ対応）
  - アクセスカウンターの数字がランダム描画されていること
  - ページ間リンク（各ページへの遷移）が正常に行われること

本ディレクトリ内のファイルをまとめ、GitHub リポジトリの `main` または `gh-pages` ブランチにPushし、Pages設定を有効化することで公開が完了します。
